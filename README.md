This is a SPA that would allow its users to manage their time better.

To access the website: https://pruki01.github.io/UCDPA-Assignment-2/

Features:
    - Creating tasks
    - Switching between different timers
    - Settings for the website
    - Form validating tasks
    - Starting, stopping and resseting the timers
